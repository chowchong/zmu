# Subgen GUI 设计规范
**用于 Google Stitch / Figma / 设计工具**

---

## 1. 窗口基本信息
- **应用名称**: Subgen - 智能字幕生成器
- **窗口尺寸**: 700px × 750px
- **背景色**: #F5F5F7 (浅灰色)

---

## 2. 配色方案

### 主色调
- **Primary (主色)**: #007AFF (蓝色)
- **Primary Hover**: #0051D5 (深蓝色)
- **Primary Pressed**: #003D99 (更深蓝色)

### 背景色
- **Background**: #F5F5F7 (浅灰背景)
- **Card**: #FFFFFF (白色卡片)
- **Card Hover**: #FAFAFA (卡片悬停)

### 文字颜色
- **Text Primary**: #1D1D1F (深灰主文本)
- **Text Secondary**: #86868B (次要文本)
- **Text Tertiary**: #C7C7CC (禁用文本)

### 状态颜色
- **Success**: #34C759 (绿色)
- **Warning**: #FF9500 (橙色)
- **Error**: #FF3B30 (红色)

### 边框颜色
- **Border**: #D1D1D6 (浅灰边框)
- **Border Focus**: #007AFF (焦点蓝色边框)

---

## 3. 布局结构（从上到下）

### 整体间距
- **外边距**: 24px (上下左右)
- **元素间距**: 20px (垂直间距)
- **卡片圆角**: 10px
- **卡片边框**: 1px solid #D1D1D6

---

## 4. 组件列表

### 📦 组件 1: 文件拖放区域（Card）
**类型**: 白色卡片容器

**内容**:
1. **拖放列表框**
   - 尺寸: 自适应宽度 × 180px (最小高度)
   - 背景: #FFFFFF
   - 边框: 2px dashed #D1D1D6 (虚线)
   - 圆角: 10px
   - 内边距: 40px 20px

2. **占位符文本**（列表为空时显示）
   - 图标: 📹 (视频emoji)
   - 文字: "拖拽视频文件到这里\n或点击下方按钮选择文件"
   - 字体大小: 14px
   - 颜色: #86868B (次要文本色)
   - 对齐: 居中

3. **文件列表项**（有文件时显示）
   - 每行显示: 🎬 [文件名.mp4]
   - 背景: #F5F5F7
   - 边框: 1px solid #D1D1D6
   - 圆角: 6px
   - 内边距: 8px 12px
   - 间距: 4px
   - 悬停效果: 背景变为 #FAFAFA，边框变为 #007AFF
   - 选中效果: 背景变为 #007AFF，文字变白色

4. **操作按钮行** (列表下方)
   - 布局: 水平排列，左对齐
   - 按钮间距: 10px
   
   **a. ➕ 添加文件 按钮**
   - 样式: 次要按钮
   - 背景: transparent
   - 文字: #007AFF
   - 边框: 1px solid #D1D1D6
   - 圆角: 8px
   - 内边距: 8px 16px
   - 字体大小: 13px
   
   **b. 🗑️ 移除选中 按钮**
   - 样式: 与添加按钮相同

**内容**:

   - 颜色: #86868B
   - 上边距: 8px

   **单选按钮列表** (垂直排列):
   
   **a. Whisper Medium (推荐) ✅**
   - 类型: 单选按钮
   - 背景: #FFFFFF
   - 边框: 1px solid #D1D1D6
   - 圆角: 8px
   - 内边距: 12px
   - 默认选中: border 变为 2px solid #007AFF
   - 悬停: 背景 #FAFAFA，border #007AFF
   
   **b. Whisper Large ✅**
   - 样式: 同上
   
   **c. Whisper Small ✅**
   - 样式: 同上
   
   **d. FunASR (实验性) ❌**
   - 样式: 同上，但禁用状态（灰色）

3. **语言选择区域**
   - 小标题: "语言:"
   - 字体大小: 12px
   - 颜色: #86868B
   - 上边距: 12px

   **单选按钮行** (水平排列):
   - **自动检测** (默认选中)
   - **中文**

   - 样式: 与模型选择按钮相同

4. **复选框**
   - 文字: "完成后自动打开编辑器"
   - 默认: 选中
   - 上边距: 8px
   - 颜色: #1D1D1F

## 5. 字体
- **字体族**: -apple-system, BlinkMacSystemFont, "SF Pro Text", "Segoe UI", sans-serif

## 6. 交互状态

### 悬停 (Hover)
- 按钮: 背景色加深
- 列表项: 背景变为 #FAFAFA，边框变蓝
- 单选按钮: 背景 #FAFAFA，边框蓝色

### 焦点 (Focus)
- 边框: 2px solid #007AFF

### 禁用 (Disabled)
- 背景: #C7C7CC
- 文字: #86868B
- 不可点击

## 7. 图标/Emoji 使用
- 📹 视频占位符

- ➕ 添加按钮
- 🗑️ 删除按钮
- ⚙️ 设置标题
- ✅ 已安装模型

## 9. 响应式注释
- 固定宽度: 700px
- 固定高度: 750px
- 不需要响应式（桌面应用）

---

## 10. 动画/过渡
- 按钮悬停: 0.2s ease
- 进度条: 平滑填充动画
- 列表项悬停: 即时边框颜色变化

## 11. 阴影效果
- 卡片阴影: 0 1px 3px rgba(0, 0, 0, 0.05)
- 按钮按下阴影: inset 0 1px 2px rgba(0, 0, 0, 0.1)
